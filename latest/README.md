# Latest OpenAPI Specifications

This directory contains the **latest generally available (GA)** OpenAPI specifications for Stripe's API, including both v1 and v2 endpoints in a single unified file.

## Files

| File | Description |
|------|-------------|
| `spec3.{json,yaml}` | OpenAPI 3.0 spec matching the public Stripe API (v1 and v2 endpoints) |
| `spec3.sdk.{json,yaml}` | Expanded OpenAPI 3.0 spec intended for Stripe-internal use |

## Other Directories

- [`/openapi/`](../openapi/)—v1-only endpoint specifications (legacy)
- [`/preview/`](../preview/)—Preview release SDK specifications
